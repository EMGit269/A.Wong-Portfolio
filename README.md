# Profolio

一个可直接部署到 GitHub Pages 的静态个人作品页。

## 文件结构

- `index.html`: 页面结构与内容
- `styles.css`: 页面样式
- `script.js`: 开场动画、流式输出与 PDF 预览交互
- `resume.pdf`: 页面内 PDF 预览附件

## 本地预览

直接用浏览器打开 `index.html` 即可预览。

## 部署到 GitHub Pages

1. 把当前目录推送到 GitHub 仓库。
2. 打开仓库的 `Settings` -> `Pages`。
3. 在 `Build and deployment` 中选择:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` 或 `master`
   - 文件夹选择 `/ (root)`
4. 保存后等待 GitHub Pages 生成公开链接。
